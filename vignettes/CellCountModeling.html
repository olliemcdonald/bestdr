Introduction

We introduce the basic workflow for running a dose response model for counting total cell counts to get the birth and death rates. We will simulate data to provide an example of how data should look for import and running in BESTDR. The resulting model will show how BESTDR can be used to distinguish cytotoxic and cytostatic effects. Second, we will simulate multiple “cell lines” to show how hierarchical modeling can be used to model multiple cell line responses simultaneously.

Single Cell Line Estimation

Data Simulation

We will generate parameters from a concentration-response curve for both the birth and death rates. That is, cell will split with rate \(b(c)\) and die with rate \(d(c)\). The net growth rate is the difference of these two, or \(g(c) = b(c) - d(c)\). The concentrations tested are at plotted as points in the figure with the smooth curve representing the smooth function for the birth (blue), death (red), and growth (purple) rates.

# Generate curve from 4-parameter logistic function for birth and death
b0 <- 1 # left asymptote
bi <- 0.2 # right asymptote
b50 <- -5 # log-dose at midpoint of birthrate (should switch to log scale)
bh <- 2 # Hill Coefficient
db <- b0 - bi # Difference in asymptotes (for parameterization)

d0 <- 0.01
di <- 0.5
d50 <- 1
dh <- 2
dd <- di - d0

b_curve <- function(lc, bi, db, bh, b50) bi + db / (1 + exp(bh * (lc - b50)))
d_curve <- function(lc, d0, dd, dh, d50) d0 + dd - dd / (1 + exp(dh * (lc - d50)))

param_truth <- data.frame(
  bi = bi, b50 = b50, bh = bh, db = db,
  d0 = d0, d50 = d50, dh = dh, dd = dd
)

crange <- c(0, 10^seq(-4, 4, 0.1))
lc_range <- log(crange)
concentrations <- c(0, 10^seq(-4, 4, 1))
log_conc <- log(concentrations)

b_truth <- b_curve(lc_range, bi, db, bh, b50)
d_truth <- d_curve(lc_range, d0, dd, dh, d50)
g_truth <- b_truth - d_truth

curve_truth <- data.frame(concentration = crange,
                          log_conc = lc_range,
                          b = b_truth, d = d_truth, g = g_truth) %>%
  pivot_longer(-c(concentration, log_conc), names_to = "Rate", values_to = "Value")

We simulate the process using the estipop package. The “rep” variable represents the replicate and the “type1” variable represents the count at a given time.

N <- 10 # Number of samples
tot_time <- 3
dt <- 0.1 # Time between measurements
times <- seq(0, tot_time, dt)
init_pop <- 200 # Number of ancestors

rate_truth <- data.frame(concentration = crange,
                         log_conc = lc_range,
                         b = b_truth, d = d_truth, g = g_truth) %>%
  filter(concentration %in% concentrations)
# Simulate process with estipop
set.seed(9283)
dat <- data.frame()
for (i in seq_along(rate_truth$concentration)) {
  params <- c(rate_truth$b[i], rate_truth$d[i])
  # time-homogeneous two-type model
  model <- estipop::process_model(
    estipop::transition(rate = estipop::rate(params[1]), parent = 1, offspring = 2), # birth
    estipop::transition(rate = estipop::rate(params[2]), parent = 1, offspring = 0) # death
  ) # clearance
  
  dat_temp <- estipop::branch(model, params, init_pop, times, N, silent = T)
  dat_temp$concentration <- concentrations[i]
  dat_temp$log_conc <- log_conc[i]
  dat <- bind_rows(dat, dat_temp)
}

Estimation

We will use a Stan file from the model library to estimate the birth and death rates. We will set up the necessary input data as a list and concatenate the prior distribution parameters for the means and standard deviations for each of the parameters to estimate.

The single type model takes the current and previous count as well as the time between the two counts in separate vectors that are the length of the number of observations (note: observations are not replicates, but the number of time points recorded over all replicates). Each observation has a concentration, convert the concentrations to an index that corresponds to a vector of unique concentrations. The concentration index is used in place of the concentrations for each observation.

That is, if we have concentrations for a set of 5 observations (0.0, 0.1, 0.1, 0.25, 0.0), then the c_idx would be (1, 2, 2, 3, 1), log_conc would be (0.0, 0.1, 0.25), and nc would be 3.

The above curves show that the simulation we generated has initial cytostatic effects noted by the decreasing birth rate followed by cytotoxic effects as the concentration increases further which is based on the increasing death rate. We were able to estimate each effect simulataneously.

Additional metrics

We can further derive additional metrics about the birth, death, or growth curves. For instance, the \(b_{50}\) and \(d_{50}\) are the concentration at which the birth and death rates reach their midpoint which are parameters of the logistic curve and can provide a measure of how cytotoxic or cytostatic an agent is.

For summary statistics, we can define the \(NG_{50}\) and \(NG_{0}\) as the concentration where the growth rate hits half it’s initial value and the concentration when the growth rate hits 0 respectively. In the latter case, this is when the cell growth starts decreasing over time.

Multiple Cell Lines (Hierarchical Model)

Data Simulation

Using data for a hierarchical model requires an additional group variable to be included. We index this the same way we index concentrations. We’ll illustrate the process for estimating from multiple cell lines by sampling parameters from distributions where we’ve specified the hyperparameters.

                inner_join(d_subset) %>%
                mutate(crange = ifelse(crange == 0, 0.000001, crange)))
}
